== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/quiveo-onsite-marketing` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin Quiveo screen to configure the plugin
4. Type in your project ID which you can see in your Quiveo account
5. Activate (set activate to yes)

Formerly to this installation you should subscribe to [Quiveo](https://www.quiveo.de)
Simply try out our 14 days trial without any risk.
